# api-predicao-ataque-cardiaco

### Você pode executar de dois modos: por máquina virtual ou por docker.
### No ambiente Windows, basta seguir os seguintes passos:
#### 1- criar uma venv:
#### python -m venv venv
#### 2- executar a venv:
#### ./venv/scripts/activate
#### 3- python manage.py runserver
